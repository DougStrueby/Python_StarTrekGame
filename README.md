# Python Star Trek Game

## Controls
- **`exit` / `q` / `quit`**: Quit the game  
- **`north` / `n`**: Move up  
- **`south` / `s`**: Move down  
- **`west` / `w`**: Move left  
- **`east` / `e`**: Move right  

## Objective
Your mission is to defeat all Klingons (`K`). Navigate to their locations using the movement controls and attack them until all are defeated.

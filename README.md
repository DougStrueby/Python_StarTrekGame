# Python_StarTrekGame

Controls: 

  exit/q/quit: quit game
  
  north/n: move up
  
  south/s: move down
  
  west/w: move left
  
  east/e: move right

Objective:

  Defeat all Klingons(K) by moving E to their locations and attacking them until all are defeated
